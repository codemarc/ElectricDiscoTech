While it is beyond the scope of this project to explain life the universe and everything is important to start with a relatively clean environment. I use Microsoft Azure to kick off a fresh instance of Windows 8.1 on a host named studio54

#Studio54

Posted by Studio 54 on Monday, October 5, 2015

Using Microsoft Remote Desktop, I login and change the background to somthing blueish. I install chrome, and git for windows accepting all defaults (next next next …).

A good code editor is the hammer in a modern web application developers toolbox. For this project I am using one of Microsoft's latest concoctions, Visual Studio Code. It's open source and forkable on github and I really like.
