Source

ElectricDiscoTech is built by standing on the shoulders of the giants, open source giants. Edt is hosted on GitHub and licensed under MIT license. You're welcome to use it for all your personal and commercial projects. To access all source files, just clone the repository on GitHub or download the latest version. In fact this file is part of the developers documentation in the dev/doc directory.

File structure

ElectricDiscoTech employs an intuitive file layout/orgainzation. By adopting it as a convention you can decrease the number of decisions that need to be made. Although it is sourced in a single tree, logically it is broken down into as:

Develop

FolderDescription
/devsource components
/dev/bower_componentssource pulled by bower
/dev/docsdocs for edt
/dev/jscompiled from typescript
/dev/lessless source
/dev/node_modulesruntime used by grunt
/dev/tstypescript source
/dev/typingstypescript definitions

Deploy

FolderDescription

/fontsfonts defintitions
